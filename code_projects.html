Projects

Meristem

A lightweight (no dependencies!) Javascript library for procedural text generation using a context-free grammar. Like template literals, or Python's string formatting, but significantly more flexible.

NPM: https://www.npmjs.com/package/meristem

Github: https://github.com/ForrestWeiswolf/Meristem

Coordin-EAT

Coordin-EAT is a mobile-first web app to ease the pain of finding the perfect spot for dinner with your geographically inconvenient friends.

Built with the NERDS stack, Coordin-EAT uses Google’s Distance matrix API and an algorithm inspired by simulated annealing to calculated convenient meeting places, and MapboxGL to display them.

I was part of the four-person team that built this webapp.

Deployed: coordin-eat.herokuapp.com

Github: github.com/Bluegrass-Grief-Owls/coordin-eat/

Our presentation of this project at FSA Demo Day: youtube.com/watch?v=DUICbdd3uTU
